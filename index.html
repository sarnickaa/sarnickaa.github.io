TicTacToe

I built a front-end JavaScript TicTacToe application (including game-logic) that connects to a third-party Rails API via AJAX. The application employs token authentication to allow users to register and signin. Users can also save and retrieve game-play statistics via CRUD requests. Includes custom logo design.
