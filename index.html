catSnack

catSnack is a pet-food preference tracker for pet owners. Keeping tabs on the type of food a pet prefers is difficult especially if pet preferences change over time. Pet food purchases are usually made in bulk quantities meaning that accidentally buying the wrong type of food for a picky pet means wasted time, money and an unhappy critter. I built a full stack application (custom JavaScript frontend and custom Rails API) full CRUD and token authentication. Users can insert and retrieve data from a postgreSQL database. I implemented 2 levels of a 1-to-many relationship. The application consumes a 3rd party API and includes custom logo design.
